# Task 4 – Sales Performance Dashboard

## Data Analyst Internship

### Objective
The objective of this project is to design an interactive dashboard using Power BI that helps business stakeholders analyze sales performance, profit, product performance, and regional trends through visualizations and filters.

---

## Tools Used
- Power BI Desktop
- Microsoft Excel
- GitHub

---

## Dataset
- Financial Sample Dataset (.xlsx)

The dataset contains information such as:
- Date
- Year
- Country
- Product
- Segment
- Sales
- Profit
- Units Sold

---

## Dashboard Features

### KPI Cards
- Total Sales
- Total Profit
- Units Sold
- Average Sales

### Charts
- Sales Trend (Line Chart)
- Product-wise Sales (Bar Chart)
- Country-wise Profit (Column Chart)
- Sales by Segment (Pie Chart)

### Interactive Slicers
- Year
- Country
- Product
- Segment

---

## Key Insights
- Total Sales exceeded 118 Million.
- Total Profit exceeded 16 Million.
- October recorded the highest sales.
- Paseo was the highest-selling product.
- France and Germany generated high profits.
- Government and Small Business segments contributed significantly to overall sales.
